Character Brain System

Right now your characters have:

{
  "name": "Aisha",
  "personality": "quiet and caring"
}

That's extremely shallow.

Real people have:

beliefs
fears
insecurities
goals
boundaries
communication style
attachment style
likes/dislikes
Build Character DNA

Instead of:

{
  "name": "Aisha",
  "personality": "quiet"
}

Move to:

{
  "name": "Aisha",

  "core_traits": [
    "introverted",
    "empathetic",
    "observant"
  ],

  "strengths": [
    "loyal",
    "patient"
  ],

  "flaws": [
    "overthinks",
    "avoids confrontation"
  ],

  "fears": [
    "abandonment",
    "rejection"
  ],

  "goals": [
    "become independent"
  ],

  "attachment_style": "avoidant",

  "communication_style": "soft and indirect"
}
